# Yet-Another-Monochrome-Icon-Theme
This is a beautiful Icon Theme made by [DIRN](https://store.kde.org/u/dirn) on KDE Store. I am just using this REPO in my automated Post Installation script. 

## Installation 

```
git clone https://www.github.com/googIyEYES/YAMIS.git && cd Yet-Another-Monochrome-Icon-Theme
tar -xzvf monochrome-icon-theme.tar.gz -C "~/.local/share/icons/"
```
